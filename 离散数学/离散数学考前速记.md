## 知识点总结

### 命题

* 只有 P ... 才 Q  => $Q \to P$
* 除非 P ... 否则 Q => $\neg P -> Q$
* 除非不P...否则不Q => $\neg\neg P \to \neg Q <=> P \to \neg Q <=> Q \to \neg P$(可以继续等价置换知道得到与选项相同的答案)



### 存在量词与全称量词的式子判断，注意这里双向箭头还是单项箭头，双向箭头表示前后互推，单项箭头表示只能前推出后

* **存在量词对析取满足分配律和逆分配律 $\exists x(P(x) \lor Q(x)) <=> \exists x P(x) \lor \exists x Q(x)$**
* **全称量词对合取满足分配律和逆分配律  $\forall x (P(x) \land Q(x)) <=> \forall x P(x) \land \forall x Q(x)$**
* **全称量词对析取不满足分配率，满足逆分配律 $\forall x P(x) \lor \forall x Q(x) => \forall x (P(x) \lor Q(x))$**
* **存在量词对合取满足分配律，不满足逆分配律$\exists x (P(x) \land Q(x)) => \exists x P(x) \land \exists x Q(x)$**

### 谓词单条件式子判断

* **谓词做前件，等价后量词要改变：$\forall x (A(x) \to B) <=> \exists x A(x) \to B$**
* **谓词做后件，等价后量词不改变：$\forall x(B \to A(x)) <=> B \to \forall x A(x)$**

* ## **对偶公式就是符号都写成反向 $A \lor B$ => $A \land B$**

### 主析取与主合取范式

* 主合取$(\lor) \land (\lor)$,并的交, 对应大项M(大M表示)，大项编码否是1, 编码作为项的指派，结果为否定
* 主析取$(\land) \lor (\land)$交的并, 对应小项m(小m表示)，小项编码肯是1，编码作为项的指派，结果为肯定
* 任意两个不同的小项合取为假，全体小项析取为真；
* 任意两个不同的大项析取为真，全体大项合取为假；

### 复杂前束范式求法

1. **换掉后面量词对应的元为其他元, 使得所有不同辖域的量词对应的元不同。不受命题变元约束的其他谓词作用域内相同名称的元也要更改元名称**
2. **转换蕴含式(单条件，双条件)为合取、析取、否定表示的形式**
3. **按表达式从左到右的顺序依次对每个量词进行辖域扩张，就是直接把该量词提到最前面，把后面的命题合并。**

### 函数

* **函数/ 映射：一对一或多对一，不能是一对多**

* **只有双射函数有逆函数**

* 单射/入射：$f: A \to B \quad y\in ranf $ 都有唯一一个A中的元素使得 $f(x) = y$  

  判断是否为单射:  定义法: 对于任意 $x_1 \neq x_2 => f(x_1) \neq f(x_2)$

* 满射：$f: A \to B \quad ranf = B $ 也就是说 B 中的任意元素都有A中元素与之对应

  判断满射：看 B 中哪些元素不属于 ranf

* 双射：同时满足入射和满射的叫双射

* 函数的复合是右复合：$F(x) \circ G(x) = G(F(x))$

* **函数的复合，函数都是单射，复合结果也是单射，同理都是双射结果也是双射，都是满射结果也是满射，但是不能逆推**

* **A(3个元素)到B(2个元素)所有不同满射函数的个数 6 ($2^3 - 2$)(有两个全射到B中某一个元素的映射)**

  **其他满射个数的例子:** 

  | A->B(元素个数) | 不同的满射个数 |
  | -------------- | -------------- |
  | 3->2           | 6              |
  | 4->2           | 14             |
  | 5->2           | 30             |

* **反函数的求法：另 y=f(x) 把 x 用 y 的方式表达，最后把 x，y 互换**

  

### **格与群**

* 格的定义：交换律 + 结合律 + 吸收律 + 幂等律(可选)
* **判断能否构成格，找任意两个结点，他们的向上或者向下延伸的交点是否唯一**
* 分配格的判断: 包含钻石格和五角格的不是分配格
* **群的定义：封闭性 + 结合律 + 单位元 + 逆元**
* **群元素的阶：设 k 是阶，元素a的阶就是 a 的 k 次群运算 = e(群的幺元)，求k，k是参与运算的a的个数如： (a * a) = e，则 k = 2。**
* **环的定义：在群的基础上增加了第二种运算，通常为乘法和加法，加法形成阿贝尔群(封闭性 + 结合律 + 单位元 + 逆元+交换律）乘法满足(封闭性 + 结合律 + 对加法满足分配律)，环首先是一个群**
* 域的定义：在环的基础上乘法也构成阿贝尔群，满足（封闭性 + 结合律 + 单位元 + 逆元 + 交换律）

* 哈密顿图**必要条件(用来证明一个图不是哈密顿图)**:

  **设无向图G=<V,E>是哈密顿图，则对于任意 $V_1 \subset V 且 V_1=\emptyset$ 均有 $W(G-V_1) \leq |V_1|$**

* **完全带权图的哈密顿回路条数 (n-1)!/2**

* ## 4阶以上完全图都是哈密顿图

  **设G是 n 阶无向简单图，若任意两个不相邻的顶点的度数之和大于等于 n-1, 则G中存在哈密顿通路(半哈密顿图)**
